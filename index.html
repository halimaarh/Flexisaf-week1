Student Score Dashboard

Code Explanation

1. Students Data: A list of students with names and scores stored in an array of objects.

2. Names: Extracted using map() to list all student names.

3. Passed: Filtered students who scored 60 or above using filter().

4. Total Score: Calculated the sum of scores using reduce().

5. Removed Student: Removed the second student (Ahmed) using splice().

6. Top Performers: Displayed students who scored above 80 using another filter().
